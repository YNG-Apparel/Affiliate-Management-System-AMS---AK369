# 7. User Flows

## Affiliate Registration Flow
1. User visits the registration page.
2. User creates an account and submits affiliate details.
3. Admin reviews and approves the affiliate.
4. Affiliate gains access to campaign and reporting tools.

## Commission Tracking Flow
1. Affiliate shares referral links.
2. A user clicks and converts.
3. The system records the event.
4. The commission is calculated and displayed in the dashboard.
