# 8. System Architecture

## Overview
This document describes the high-level architecture of the AMS.

## Architecture Summary
- Frontend applications for admin and affiliate users.
- Backend services handling business logic and APIs.
- Database for storing users, affiliates, campaigns, referrals, and payouts.
- Infrastructure for deployment, monitoring, and scaling.

## Architectural Style
The system is planned as a modular, service-oriented platform with clear separation between presentation, business logic, and data storage.
