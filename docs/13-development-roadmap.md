# 13. Development Roadmap

## Overview

Implementation follows the phased, three-month plan defined in the project proposal.
Each month delivers a usable increment, with TikTok Shop as the first integrated platform.
Shopee and Tokopedia follow in later phases.

> Parallel dependency: TikTok Shop and Shopee Open Platform developer/partner approval can
> take 2–4 weeks. Registration must start alongside development to avoid delaying go-live.

---

## Month 1 — MVP (Operational)

Goal: a fully operational core system for day-to-day affiliate operations.

- Project setup, authentication, roles, and permissions
- Manager Dashboard shell and Affiliator App (PWA) shell
- Affiliate management for 500+ accounts (create, approve, suspend, search, filter)
- TikTok Shop integration — OAuth account connection and real-time sales tracking
- Automatic order attribution to the originating affiliate
- Automatic payroll: base salary (Internal), sales bonus, tier multiplier, approval workflow
- Bulk transfer export for internet banking
- Central content distribution: content library, distribution by city, auto yellow-cart attach
- Caption templates and scheduled posting
- Per-account posting status monitor

---

## Month 2 — Enhancement

Goal: gamification, self-service onboarding, and broader tracking.

- Tier system and gamification (Starter / Advanced / Elite, leaderboard, performance alerts)
- Digital onboarding for freelance affiliates (self-registration, KYC, digital contract)
- Freelance → Internal promotion workflow
- Voucher system — unique codes per affiliate for Shopee/Tokopedia tracking
- Per-affiliate product assignment
- Shopee integration groundwork (multi-store view, voucher tracking)
- Notifications (FCM push, WhatsApp/SMS onboarding invites)

---

## Month 3 — Advanced & Go-Live

Goal: complete the multi-platform picture, secure content, and harden for production.

- Full Shopee integration (multi-store GMV aggregation)
- TikTok LIVE monitoring and gift/diamond bonus tracking
- Content security: automatic watermarking, direct upload, film/drama blocking for freelancers
- Revenue reconciliation (fees received vs. paid out)
- Reporting suite (sales, revenue, affiliate ranking, payroll, product performance)
- Performance optimization, security hardening, and production go-live

---

## Later Phases

- Tokopedia integration (Phase 3)
- Meta / Facebook content distribution via Graph API
- Advanced analytics and automation
